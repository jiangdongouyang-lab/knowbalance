# TypeScript Generics Code Lab

Complete these exercises to learn TypeScript generics fundamentals.

## Exercises

### Exercise 1: Identity Function
Create a generic identity function that accepts any type and returns the same value.

### Exercise 2: Array Utilities
Build generic functions to find elements in arrays and filter by type.

### Exercise 3: Stack Implementation
Implement a generic Stack data structure with push, pop, and peek operations.

## Getting Started

```bash
npx ts-node exercise-01.ts
npx ts-node exercise-02.ts
npx ts-node exercise-03.ts
```
