# K002 变量与赋值

## 1. 核心定义

Python 使用 = 进行变量赋值。

## 2. 必须掌握

- Python 使用 = 进行变量赋值。
- 变量名用于引用程序中的数据。
- 变量可以被重新赋值，新值会覆盖旧绑定。

## 3. 示例

### 更新变量值

```python
age = 18
age = age + 1
print(age)
```

变量可以先保存旧值，再通过赋值语句更新为新值。
