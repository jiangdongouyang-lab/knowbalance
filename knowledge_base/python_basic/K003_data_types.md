# K003 基本数据类型

## 1. 核心定义

int 表示整数，float 表示小数。

## 2. 必须掌握

- int 表示整数，float 表示小数。
- str 表示字符串文本。
- bool 表示 True 或 False 两种逻辑值。

## 3. 示例

### 识别基础类型

```python
score = 95
name = "Li Hua"
passed = True
print(type(score), type(name), type(passed))
```

int、str、bool 表示不同类型的数据，后续运算前应先确认类型。
