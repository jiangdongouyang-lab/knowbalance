# K008 while 循环

## 1. 核心定义

while 会在条件为 True 时重复执行代码块。

## 2. 必须掌握

- while 会在条件为 True 时重复执行代码块。
- while 循环必须更新循环条件以避免死循环。
- while 适合重复次数不确定但停止条件明确的任务。
