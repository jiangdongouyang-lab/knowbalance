# K009 列表

## 1. 核心定义

列表可用于保存多个有序元素。

## 2. 必须掌握

- 列表可用于保存多个有序元素。
- 列表元素可以通过索引访问。
- append 可向列表末尾添加元素。

## 3. 示例

### 保存多名学生成绩

```python
scores = [80, 90, 75]
scores.append(88)
print(scores)
```

列表适合保存多个有序数据，append 可追加新元素。
