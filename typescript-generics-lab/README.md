# TypeScript Generics Lab

A hands-on learning experience for JavaScript developers transitioning to TypeScript.

## Exercises

### Exercise 1: Basic Generic Functions
- Understand what generics are and why they're useful
- Write your first generic function
- See how generics preserve type information

### Exercise 2: Generic Interfaces & Types
- Use generics in interfaces and type aliases
- Create flexible, reusable data structures
- Understand type constraints

### Exercise 3: Type Constraints & Common Patterns
- Use `extends` to constrain generic types
- Understand keyof and mapped types with generics
- Apply real-world generic patterns

## Getting Started

1. Open the exercise files in `exercises/` directory
2. Complete the tasks marked with comments
3. Uncomment test cases to verify your implementation
4. Check solutions in `solutions/` directory if needed

## Requirements

- TypeScript 4.0+
- Basic JavaScript knowledge
- Familiarity with TypeScript basics (types, interfaces)
